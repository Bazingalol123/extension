Brave Setup — Hide Native Tab Bar
💡
For the full Arc experience, hide Brave's native horizontal tab bar. This requires two steps.
1
Right-click any tab → Use Vertical Tabs
Or enable via brave://flags/#brave-vertical-tabs → set to Enabled → relaunch
2
Once vertical tabs panel appears, right-click the vertical sidebar → Hide tab bar
The horizontal bar disappears. Our extension's side panel takes its place.
3
Go to brave://settings/appearance → turn off "Show bookmarks bar" for maximum clean space.
Behavior
Open sidebar on browser start
Automatically open the Arc side panel when Brave launches
Show muted tab indicator
Display a mute icon on tabs with audio muted
Group new tabs by space automatically
New tabs opened inherit the currently active space
